# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-18

### Added
- Complete governance course with 6 labs covering Unity Catalog, MLOps, monitoring, and secrets
- 9 example categories covering all course lessons
- Course outline and capstone project documentation

### Examples
- `unity/` - Unity Catalog: create catalogs, schemas, tables, and volumes
- `access-control/` - GRANT/REVOKE and least-privilege permission model design
- `lineage/` - Querying lineage and audit data from Unity Catalog system tables
- `repos/` - GitHub Actions workflow for Databricks Repos integration
- `cicd/` - Multi-environment CI/CD pipeline for notebook promotion
- `mlops/` - MLflow experiment tracking, model registration, and registry promotion
- `monitoring/` - Lakehouse Monitoring table monitors and anomaly alerts
- `ml-monitoring/` - Inference table drift detection and retraining triggers
- `secrets/` - Databricks secret scopes and cloud vault integration
